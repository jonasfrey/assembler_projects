# installation
this is a handy script to create a file structure for assembler projects/scripts 

```bash
#create file structure
./create_new_assembly_script.sh my_new_test_script 
# cd into folder
cd my_new_test_script 
# make and execute 
./make_make_and_execute.sh
```
